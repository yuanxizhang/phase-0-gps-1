# Awesome GIT Commands 

## Create
* git init
* git clone URL

## Local Changes
* git status
* git diff
* git add .
* git add <file>
* git commit -m "Changes made"

## Branches
* git branch
* git checkout -b <branch>

## Commit History
* git log

## Updates
* git pull <remote> <branch>
* git push <remote> <branch>

## merge
* git merge <branch>
